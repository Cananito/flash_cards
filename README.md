# flash_cards

## Dev Set Up

Install Django

```
$ python3 -m venv venv/
$ source venv/bin/activate
$ python3 -m pip install Django
$ deactivate
```

Install TypeScript:

```
npm install
```

Note: TypeScript was set up in this repo with:

```
npm install typescript --save-dev
```
